# 枚举类型

## 数字枚举

**枚举成员值默认从0开始递增**

```javaScript
// 数字
enum Role {
    Reporter = 1,
    Developer,
    Maintainer,
    Owner,
    Guest
}
```
## 字符串枚举

**不支持反向映射**

```javaScript
    // 字符串枚举
enum Message {
    Success = '恭喜你，成功了',
    Fail = '抱歉，失败了'
}
```
## 异构枚举

```javaScript
    
// 异构枚举
enum Answer {
    N,
    Y = 'Yes'
}
```

## 枚举成员

```JavaScript
// 枚举成员
// Role.Reporter = 0
enum Char {
    // const member
    a,
    b = Char.a,
    c = 1 + 3,
    // computed member
    d = Math.random(),
    e = '123'.length,
    f = 4
}

```
